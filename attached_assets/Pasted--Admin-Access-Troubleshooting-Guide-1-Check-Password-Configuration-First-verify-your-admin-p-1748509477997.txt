# Admin Access Troubleshooting Guide

## 1. Check Password Configuration

First, verify your admin password is correctly set:

```typescript
// In your server/index.ts or environment variables
const ADMIN_PASSWORD = process.env.ADMIN_PASSWORD || 'your-default-password';
```

**Actions to take:**
- Check if `ADMIN_PASSWORD` environment variable is set in Replit Secrets
- Try using the default password if one was hardcoded
- Ensure the password comparison is case-sensitive

## 2. Clear Browser Data

Admin login issues often stem from cached data:

**Steps:**
1. Open browser developer tools (F12)
2. Go to Application/Storage tab
3. Clear localStorage and sessionStorage
4. Clear cookies for your site
5. Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)

## 3. Check Network Requests

Monitor what's happening when you try to login:

**Debug steps:**
1. Open browser developer tools
2. Go to Network tab
3. Try logging in
4. Look for failed requests (red entries)
5. Check response codes and error messages

## 4. Verify Admin Route

Ensure your admin route is properly configured:

```typescript
// Check if this exists in your server
app.post('/api/admin/login', (req, res) => {
  const { password } = req.body;
  if (password === ADMIN_PASSWORD) {
    // Success logic
  } else {
    // Error handling
  }
});
```

## 5. Console Errors

Check browser console for JavaScript errors:

**Steps:**
1. Open developer tools (F12)
2. Go to Console tab
3. Look for red error messages
4. Share any authentication-related errors

## 6. Database Connection

If admin data isn't loading, check database connectivity:

```typescript
// Verify database connection in server logs
console.log('Database connected:', db.readyState);
```

## 7. Common Fixes to Try

### Fix 1: Reset Authentication State
```javascript
// In browser console, run:
localStorage.clear();
sessionStorage.clear();
location.reload();
```

### Fix 2: Check Environment Variables
In Replit:
1. Go to Secrets tab
2. Verify `ADMIN_PASSWORD` is set
3. Restart the application

### Fix 3: Bypass Authentication (Temporary)
```typescript
// Temporarily comment out authentication check
// if (password === ADMIN_PASSWORD) {
  res.json({ success: true });
// }
```

### Fix 4: Add Debug Logging
```typescript
app.post('/api/admin/login', (req, res) => {
  console.log('Login attempt:', req.body);
  console.log('Expected password:', ADMIN_PASSWORD);
  console.log('Received password:', req.body.password);
  // ... rest of login logic
});
```

## 8. Test Admin Access

Try accessing admin through different methods:

1. **Direct URL**: `your-site.replit.dev/admin`
2. **Incognito Mode**: Test in private browsing
3. **Different Browser**: Try Chrome, Firefox, etc.

## Next Steps

If none of these work, please share:

1. **Exact error messages** from browser console
2. **Server logs** from Replit console
3. **Network request details** showing failed calls
4. **Current admin password** you're trying to use
5. **Screenshots** of the login screen or errors

This information will help identify the specific issue preventing admin access.
